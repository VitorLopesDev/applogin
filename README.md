# AppLogin

## Descrição

O **AppLogin** é um projeto desenvolvido em **Java com Spring Boot**, com foco em implementar um sistema básico de autenticação de usuários. O objetivo principal do projeto é aplicar conceitos fundamentais de **backend**, organização de código e boas práticas de desenvolvimento em Java.

Este repositório faz parte do meu processo de aprendizado em **Ciência da Computação**, com foco em me tornar um desenvolvedor **Java Back-end**.

---

## Tecnologias Utilizadas

* Java
* Spring Boot
* Maven
* HTML
* CSS
* Docker
* mySql

## Como Executar o Projeto Localmente

### Pré-requisitos

* Java JDK 17 (ou compatível)
* Maven (ou Maven Wrapper incluído no projeto)

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/VitorLopesDev/applogin.git
```
2. Acesse a pasta do projeto;

3. Ajuste localmente o mySql ou suba um conteiner com mySql já configurado;

4. Execute a aplicação;

5. Acesse no navegador:

```
http://localhost:8080
```
